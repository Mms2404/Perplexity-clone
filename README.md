# Perplexity clone 
The app will :
1 . Do a Tavily + Trafilatura search
2 . Get text + Gemini summary
3 . Also gets 4–6 image URLs via Pixabay
4 . Sends all of them to Flutter via WebSocket
5 . Shows everything — beautifully
